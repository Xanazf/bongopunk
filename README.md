## рҹҢҲ Dual Modes

- **Baryon** (Light Mode): Clean light theme with darker, muted accent colors
- **Antibaryon** (Dark Mode): Synthwave dark theme with bright neon accents

## рҹҺҜ Supported Applications

- **Neovim** - Full colorscheme with syntax highlighting
- **Yazi** - File manager flavor with custom icons
- **Kitty** - Terminal emulator theme
- **Fish** - Shell prompt and syntax highlighting
- **Tmux** - Terminal multiplexer styling
- **Alacritty** - GPU-accelerated terminal theme
- **Rofi** - Application launcher theme
- **Dunst** - Notification daemon styling
- **QT Applications** - System-wide QT styling
- **GTK Applications** - GNOME/GTK theme support

## рҹҡҖ Quick Install

```bash
# Clone the repository
git clone https://github.com/Xanazf/bongopunk.git ~/.config/bongopunk

# Run the installation script
cd ~/.config/bongopunk
./extras/scripts/install.sh
```

## рҹҺЁ Color Palette

### Antibaryon (Dark Mode)
- **Background**: `#262335` (Dark purple-gray)
- **Foreground**: `#8BA7A7` (Light blue-gray)
- **Accents**: Bright neon colors (cyan, green, orange, pink, purple, red, yellow)

### Baryon (Light Mode)
- **Background**: `#FAFAFA` (Light gray)
- **Foreground**: `#2D2D2D` (Dark gray)
- **Accents**: Darker, muted versions of the neon colors
